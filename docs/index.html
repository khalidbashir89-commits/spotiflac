How it works
Up and running
in minutes
1

Install Prerequisites

Install Python, gamdl (pip install gamdl), and ffmpeg. These handle the Apple Music decryption and audio conversion.

2

Export Apple Music Cookies

Log into music.apple.com in Chrome, then export cookies with the "Get cookies.txt LOCALLY" extension. Drop the file in SpotiFLAC's setup page.

3

Launch SpotiFLAC

Double-click SpotiFLAC.exe — the app opens in its own window. No browser, no terminal required.

4

Search & Download

Search for any song or paste an Apple Music URL. Select tracks, hit download — files appear in Downloads/FLAC Music/ organized by collection.

Setup Guide
Apple Music setup

One-time setup to unlock lossless Apple Music downloads.

1

Install Python & gamdl

Download Python from python.org, then run: pip install gamdl

2

Install ffmpeg

Download from ffmpeg.org and add it to your PATH, or place ffmpeg.exe in %USERPROFILE%\.spotiflac\

3

Export Apple Music Cookies

Log into music.apple.com in Chrome or Edge. Install the "Get cookies.txt LOCALLY" extension and export cookies for music.apple.com.

4

Upload Cookies in SpotiFLAC

Open SpotiFLAC, click Apple Music Setup (or go to /apple-setup), and drop your cookies.txt file. Done — lossless downloads are now unlocked.
